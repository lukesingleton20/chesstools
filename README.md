# chesstools

<!-- badges: start -->
<!-- badges: end -->

Chesstools provides the user with functions to download games of an
individual player from both of the popular chess websites.

## Installation

You can install the development version of chesstools from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("lukesingleton20/chesstools")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(chesstools)
retrieve_lichess_games(player = "Lichess Username")
```
